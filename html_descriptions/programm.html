Пусть F(z)— многочлен, z—комплексное число. Рассмотрим следующую последовательность:
z0, z1 = F(z0), z2 = F(F(z0)) = F(z1),
z3=F(F(F(z0))) = F(z2) ...
Множество Мандельброта — это множество всех c ϵ ₵, при которых {zn} для F(z)=z2 +c и z0 не стремится к бесконечности.
Множество Жюлиа — это множество тех значений z0, для которых поведение последовательности {zn} может резко меняться при сколь угодно малых изменениях z0.
Данная программа строит множество Мандельброта по заданной функции и для любой выбранной Вами точки рисует множество Жюлиа.
